# Celeste's Canvas

## **Overview**

**Celeste's Canvas** is a digital sketchbook application built using **Python**, **Tkinter**, and **Pillow**. It allows users to draw freely on a digital canvas, organize artwork across multiple pages, and save their creations as PNG images.

This project was developed to strengthen my understanding of **GUI development**, **event-driven programming**, and **image processing** while combining programming with digital art.

---

## **Features**

- **Freehand Drawing**
- **Multiple Brush Colors**
- **Custom Color Picker**
- **Eraser Tool**
- **Adjustable Brush Sizes**
- **Undo Last Stroke**
- **Clear Canvas**
- **Save Artwork as PNG**
- **Multi-Page Sketchbook**
- **Page Flip**
- **Background Image Support**

---

## **Technologies Used**

- **Python**
- **Tkinter**
- **Pillow (PIL)**

---

## **Installation**

Clone the repository:

```bash
git clone https://github.com/kashaf-fatima794/Celeste-Canvas.git
```

Install the required package:

```bash
pip install pillow
```

Run the application:

```bash
python sketchbook_code.py
```

If the `python` command is unavailable:

```bash
py sketchbook_code.py
```
